# ProjetLO07

##doc git basique:

`git status` sert a comparer les différences entre les fichiers locaux et sur github  
`git add cheminDuFichier` sert à ajouter un fichier au prochain commit (`git add .` pour tout ajouter en même temps)  
`git commit -m "description"` sert à "valider" les changements ajoutés avec le add  
`git push` sert a uploader le contenu des commits sur github
